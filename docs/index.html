rdx-example

A simple example app to demonstrate "rdx" which aims to be "like Redux, but smaller" and with a much simpler API to reduce the amount of boilerplate code required.

The entire app is just 3.1 kb minified and gzipped*, but includes:

  • Redux-like state store (actions + reducers + middleware)
  • Redux Dev-Tools integration
  • State persistence / hydration using localStorage
  • Async action handling (thunks middleware)
  • Routing and parameter extraction
  • UI components bound to state store

* Safari currently requires a 1kb polyfill for EventTarget support

Each of the links below loads a separate view, using the router. The todos link loads some remote JSON data with logic to prevent re-loading if fetched (including when going between the list & detail views). Data is persisted to localStorage (so the counter is saved), clear that out to trigger the data reloading.
